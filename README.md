# Basic Codeigniter 3.1.13 for PHP 8.2.12

Ini adalah basic Codeigniter 3.1.13 untuk PHP 8.2.12 tanpa error
Untuk tidak menggunakan database, harap ubah pengaturan
\```bash
$autoload['libraries'] = array('database', 'session');
\```

menjadi

\```bash
$autoload['libraries'] = array('');
\```
